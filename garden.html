Generative 3D Garden

Interactive 3D Garden built with p5.js, using parametric equations, fractals, and trigonometry
A project by Sara Saju

Overview

This project is a generative 3D garden built with p5.js WEBGL. It features trees, flowers, bushes, butterflies, clouds, and a winding stone path, all positioned dynamically using mathematical functions.

User interactions, including rotation and zoom, allow an immersive exploration of the garden.

Elements Included

  • Trees - Branches and petals
  • Flowers
  • Bushes
  • Pathway of stones
  • Butterflies
  • Plane of grass

Code & Explanation

The garden uses x, y, z coordinates to position elements. Trigonometry, parametric equations, and fractals create natural patterns and realistic growth.

  • Tree Branching: Recursive growth using x = x + L cos(θ) and y = y - L sin(θ), scaled with Lₙ = 0.75 Lₙ₋₁ to create natural branch shrinkage.
  • Flowers: Petals placed using parametric equations x = r cos(θ), y = r sin(θ), with Golden Angle θ = i × 137.5° to prevent overlap.
  • Butterflies: Sine-based wing motion flapAngle = sin(t) × 10°; symmetry maintained via x = r cos(a), y = r sin(a).
  • Clouds & Leaves: Julia and Mandelbrot fractals create naturally distributed cloud and leaf clusters.
  • Stone Path: Elliptical placement x = r cos(θ) + random(-40,40), y = r sin(θ) + random(-40,40) ensures a natural winding path.

Challenges Faced

  • Balancing visual complexity with performance in WEBGL.
  • Ensuring natural motion without repetitive or overly symmetrical patterns.
  • Creating depth perception using scaling, gradients, and perspective instead of heavy lighting/shadows.
